Селекторы

При помощи различных селекторов можно выбрать один или группу элементов. Селекторов довольно много, поэтому рассмотрим основные, которые могут применяться каждый день.

Селектор элемента (тега)​

Описывает к какому типу элементов будет применено CSS-правило. Селектор элемента недостаточно специфичный для того, чтобы стилизировать отдельные элементы, поэтому чаще всего применяется для оформления всех тегов одного типа на странице. Например если у всех ссылок в документе нужно убрать подчёркивание.

/* Применится ко всем абзацам на странице */
p {
  font-size: 24px;
}

/* Применится ко всем ссылкам на странице */
a {
  text-decoration: none;
}

Селектор класса​

Основной селектор в современной разработке. Используется в связке с глобальным атрибутом class. Имя класса задаётся только на английском языке, и это обязательно существительное, описывающее что это за элемент. Для пополнения словарного запаса используйте справочник, словарь терминов и сокращения.

<h1 class="title">Привет, я Mango.</h1>
<p class="text">
  Добро пожаловать на мою личную страницу. Тут можно посмотреть
  <a class="link" href="">проекты</a>.
</p>

В CSS используется селектор класса, чтобы задать стили для одного и более элементов с одинаковым значением атрибута class (именем класса). В селекторе, перед именем класса ставится точка ..

/* Применится ко всем тегам с классом title */
.title {
  font-weight: 500;
}

/* Применится ко всем тегам с классом text */
.text {
  color: brown;
  font-size: 18px;
}

/* Применится ко всем тегам с классом link */
.link {
  text-decoration: none;
}
Внимание

В имени класса используются только маленькие буквы и тире. Например, list, list-item, logo-image и т. д. Подчёркивания list_item, заглавные буквы ListItem или цифры listitem27 считаются дурным тоном.

Композиция классов​

У элемента может быть несколько классов, тогда они записываются через пробел.

<element class="value1 value2 value3"></element>

Это можно применять для комбинирования и повторного использования стилей. Создадим разметку и стили для компонента оповещений.

<p class="alert success">Пополнение счёта выполнено</p>
<p class="alert warning">Внимание, будут изменены тарифы</p>
<p class="alert error">Ошибка проведения транзакции</p>

В стилях описываем несколько правил, общие для всех оповещений и специфические для каждого типа.

/* Общие стили для всех типов оповещений */
.alert {
  font-size: 24px;
  font-weight: 500;
}

/* Специфические стили для каждого типа */
.success {
  color: green;
}

.warning {
  color: orange;
}

.error {
  color: red;
}

Вот рабочий пример.

Селектор идентификатора​

Как и с классом, тегам можно задавать глобальный атрибут id. Отличие в том, что значение id должно быть уникальным на странице, то есть не может быть двух элементов с одинаковыми значениями идентификатора.

<h1 id="title">Заголовок страницы</h1>

В CSS-селекторе, перед именем идентификатора, ставится символ решетки #.

#title {
  font-weight: 500;
  color: orange;
}
Полезно

Не используйте идентификаторы для оформления. Правило уникальности значения делает повторное использование стилей невозможным. Идентификаторы предназначены для «якорей».

Селектор потомка​

Селектор потомка, или контекстный селектор, применяет стили к детям и потомкам элемента при любой глубине вложенности.

<a href="">Читайте наш блог!</a>

<ul class="social-links">
  <li>
    <a href="">Twitter</a>
  </li>
  <li>
    <a href="">Instagram</a>
  </li>
  <li>
    <a href="">Facebook</a>
  </li>
</ul>

Например вместо того, чтобы задавать стиль всем ссылкам, нужно изменить стили только тех, которые находятся внутри списка с классом social-links.

/* ❌ Применится ко всем ссылкам в документе */
a {
  color: blue;
  text-decoration: none;
}

/* ✅ Применится только к ссылкам, которые лежат внутри тега с классом social-links */
.social-links a {
  color: blue;
  text-decoration: none;
}

Селектор ребёнка​

Ещё один распространённый селектор, который позволяет выбрать только непосредственного ребёнка внутри родительского элемента. Используем разметку для многоуровневого меню магазина, в котором необходимо стилизировать теги <li> в списке с классом menu, но не submenu.

<ul class="menu">
  <li>
    Комьютеры и комплектующие
    <ul class="submenu">
      <li>Процессоры</li>
      <li>Мониторы</li>
      <li>Видеокарты</li>
    </ul>
  </li>
  <li>
    Бытовая техника
    <ul class="submenu">
      <li>Холодильники</li>
      <li>Телевизоры</li>
      <li>Стиральные машины</li>
    </ul>
  </li>
  <li>
    Товары для дома
    <ul class="submenu">
      <li>Кресла</li>
      <li>Матрасы</li>
      <li>Электрокамины</li>
    </ul>
  </li>
</ul>

Если использовать селектор потомка (контекстный), то CSS-правило применится ко всем тегам <li>, и красная рамка будет у каждого.

/* ❌ Не решает задачу */
.menu li {
  border: 1px solid tomato;
}

Селектор ребёнка помогает решить эту проблему, выбрав только те <li>, которые являются детьми (первая вложенность) списка ul.menu.

/* ✅ То что нужно */
.menu > li {
  border: 1px solid tomato;
}

Селектор атрибута​

Позволяет применить стили к элементам с определённым атрибутом или его значением.

элемент[атрибут]​

Применит стили ко всем элементам у которых есть указанный атрибут.

<a href="https://htmlreference.io" title="Справочник тегов">
  htmlreference.io
</a>

<a href="https://cssreference.io" title="Справочник стилей">
  cssreference.io
</a>

<a href="https://www.youtube.com">YouTube </a>

Выберет все ссылки с атрибутом title.

a[title] {
  color: teal;
}

элемент[атрибут="значение"]​

Применит стили ко всем элементам, у которых есть атрибут с указанным значением.

<a href="https://htmlreference.io" title="Справочник тегов">
  htmlreference.io
</a>

<a href="https://cssreference.io" title="Справочник стилей">
  cssreference.io
</a>

<a href="https://www.youtube.com">YouTube</a>

Выберет все ссылки с атрибутом href и соотвествующим значением.

/* Непривычное форматирование фигурных скобок это нормально */
a[href="https://htmlreference.io"]
{
  color: teal;
}

a[href="https://cssreference.io"]
{
  color: orange;
}

a[href="https://www.youtube.com"]
{
  color: tomato;
}
