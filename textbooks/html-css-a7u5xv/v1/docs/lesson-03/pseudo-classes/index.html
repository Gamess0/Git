Псевдоклассы состояния

Селекторы состояния используются для применения стилей к интерактивным элементам при определённом событии, например наведении курсора по ссылке или фокусировке с клавиатуры.

Псевдокласс определяет конкретное состояние элемента и прикреплён к селектору тега, класса и т. п. Пробела между селектором и псевдоклассом нет, чтобы показать, что они связаны друг с другом. Если добавить пробел, то псевдокласс применится ко всем элементам в документе.

селектор:псевдокласс {
  /* Свойства */
}

Псевдокласс :hover​

Активируется когда курсор мыши находится в пределах элемента, например при наведении мыши на ссылку или любой другой элемент.

<p>
  Отличную шпаргалку по тегам можно найти
  <a class="link" href="https://htmlreference.io">по ссылке</a>.
</p>

Добавим ссылке базовый стиль и стиль при наведении.

.link {
  color: teal;
}

.link:hover {
  color: tomato;
}

Псевдоклассы можно применять к любым селекторам, в том числе контекстным.

<ul class="social-links">
  <li><a class="link" href="https://twitter.com">Twitter</a></li>
  <li><a class="link" href="https://www.instagram.com">Instagram</a></li>
  <li><a class="link" href="https://www.facebook.com">Facebook</a></li>
</ul>

Добавим ссылке внутри списка базовый стиль и стиль при наведении.

.social-links .link {
  color: teal;
}

.social-links .link:hover {
  color: tomato;
}
Полезно

Такой селектор читается как «При ховере по элементам с классом link, которые находятся внутри элемента с классом social-links». То есть читаются селекторы справа налево.

Псевдокласс :focus​

Активируется когда интерактивный элемент (ссылка, кнопка, поле формы) получает фокус при навигации по странице клавиатурой (клавишей Tab).

<ul class="social-links">
  <li><a class="link" href="https://twitter.com">Twitter</a></li>
  <li><a class="link" href="https://www.instagram.com">Instagram</a></li>
  <li><a class="link" href="https://www.facebook.com">Facebook</a></li>
</ul>

Добавим ссылке внутри списка базовый стиль и стиль при фокусе.

.social-links .link {
  color: teal;
}

.social-links .link:focus {
  color: orange;
}

Для того, чтобы учесть наведение и фокусировку, стоит стилизировать сразу оба состояния. Это делается простым перечислением селекторов.

.social-links .link {
  color: teal;
}

/* Два и более селекторов разделяются запятой */
.social-links .link:hover,
.social-links .link:focus {
  color: orange;
}

Псевдокласс :active​

Происходит при активации элемента. Например, ссылка становится активной, если навести на неё курсор и щёлкнуть мышкой. Несмотря на то, что активным может стать практически любой элемент, псевдокласс :active используется в основном для ссылок и кнопок.

<ul class="social-links">
  <li><a class="link" href="https://twitter.com">Twitter</a></li>
  <li><a class="link" href="https://www.instagram.com">Instagram</a></li>
  <li><a class="link" href="https://www.facebook.com">Facebook</a></li>
</ul>
.social-links .link {
  color: teal;
}

.social-links .link:hover,
.social-links .link:focus {
  color: orange;
}

.social-links .link:active {
  color: red;
}

Псевдокласс :visited​

Этот псевдокласс активируется на ссылках, которые уже были посещены. По умолчанию ссылки отображаются синими и при посещении становятся фиолетовыми.

<ul class="social-links">
  <li><a href="https://twitter.com" class="link">Twitter</a></li>
  <li><a href="https://www.instagram.com" class="link">Instagram</a></li>
  <li><a href="https://www.facebook.com" class="link">Facebook</a></li>
</ul>
.social-links .link {
  color: teal;
}

.social-links .link:hover,
.social-links .link:focus {
  color: orange;
}

.social-links .link:active {
  color: red;
}

.social-links .link:visited {
  color: green;
}

Ховер и таблицы​

Псевдокласс :hover не обязательно должен применяться к ссылкам, его можно добавлять и к другим элементам документа. Например, к строкам или ячейкам таблицы, чтобы они меняли цвет фона при наведении курсора мыши.

<table class="schedule">
  <thead>
    <tr>
      <th>Номер</th>
      <th>Маршрут</th>
      <th>Время в пути</th>
    </tr>
  </thead>

  <tbody class="schedule-body">
    <tr>
      <td>433</td>
      <td>Киев - Ковель</td>
      <td>5 часов</td>
    </tr>
    <tr>
      <td>701</td>
      <td>Харьков - Днепр</td>
      <td>7 часов</td>
    </tr>
    <tr>
      <td>258</td>
      <td>Львов - Одесса</td>
      <td>4 часа</td>
    </tr>
  </tbody>
</table>

Добавим базовое оформление таблицы и стили для ховера по строкам.

.schedule-body > tr:hover {
  background-color: tomato;
  color: white;
}

/* Оформление таблицы */
.schedule {
  border-collapse: collapse;
}

.schedule th,
.schedule td {
  padding: 8px;
  border: 1px solid #212121;
}

Вот рабочий пример.

Свойство cursor​

Устанавливает тип указателя мыши, когда он находится в пределах элемента. Вид курсора зависит от операционной системы и её настроек. Менять тип курсора просто так плохая идея, это только запутает пользователя, когда вместо привычной руки или лупы он увидит что-то другое. В большинстве случаев лучше оставить всё как есть.

Наводите мышкой на текст (значения свойства) и курсор будет изменять свой вид.
